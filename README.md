# Retail Sales Analysis

This project focuses on loading, cleaning, and analyzing a retail sales dataset. It includes descriptive statistics, time series analysis, customer and product analysis, and visualizations. Finally, it provides actionable recommendations based on the exploratory data analysis (EDA).
